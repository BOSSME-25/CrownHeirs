# Crown Heirs — Public Site

This repo holds the **public, customer-facing** Crown Heirs site
(`crown-heirs-deploy/`).

The internal employee Team Hub previously lived here in
`crown-heirs-intranet/`. It has moved to its own **private** repository,
`BOSSME-25/CrownTeam`, so internal code is no longer publicly readable.
All Team Hub development happens there now — the Vercel `crown_team_hub`
project deploys from that repo.
